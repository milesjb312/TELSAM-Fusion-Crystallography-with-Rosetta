# TELSAM-Fusion-Crystallography-with-Rosetta
This repository is the basis for Rosetta and PyRosetta scripts that enable TFC (TELSAM Fusion Crystallography). It will allow users to create TELSAM fusion proteins and test linker variants for optimal crystallizability.
The main program, start_TELSetta.py, takes two arguments: the TELSAM mutant you desire to use (currently only 1TEL is allowed) and the PDB of the client you intent to fuse (either a file in your system or the PDB's code from the RCSB PDB). The program takes a TELSAM monomer with a helical linker at the end, extends the linker, and fuses the client to the linker at the client's first 5-amino acid length helix. It then creates, docks, and scores the TELSAM fusion polymer by shrinking the unit cell, rotating the polymer, and making symmetric poses of the fusion iteratively using PyRosetta. It returns the poses with the lowest scores and charts the scores versus their unit cell size and degree of polymer rotation. The program then repeats this process for a total of 14 different linker lengths.
The next big update will introduce the option to test various helical rises of the TELSAM fusion polymer. This will include both changing the unit cell and rotating the monomers within the polymer to maintain the polymer contact points in coincidence.
